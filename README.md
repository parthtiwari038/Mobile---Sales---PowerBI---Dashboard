# 📱 Mobile Sales Power BI Dashboard

## 📌 Project Overview

This project presents an interactive Mobile Sales Dashboard developed using Microsoft Power BI. The dashboard helps analyze sales performance, customer purchasing trends, payment methods, and geographical sales distribution. It enables businesses to make data-driven decisions through interactive visualizations and key performance indicators (KPIs).

---

## 🎯 Objectives

- Analyze overall mobile sales performance.
- Track monthly sales trends.
- Identify top-performing mobile brands and models.
- Analyze customer payment preferences.
- Compare sales across different cities.
- Provide interactive filtering for deeper analysis.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

---

## 📊 Key Performance Indicators (KPIs)

- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales Value

---

## 📈 Dashboard Features

- Interactive KPI Cards
- Monthly Sales Trend Analysis
- Sales by Mobile Brand
- Sales by Mobile Model
- Payment Method Analysis
- Customer Ratings Analysis
- City-wise Sales Map
- Dynamic Slicers and Filters
